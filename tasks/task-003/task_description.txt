- Add taskService.removeTagFromTask method
- Expose tag methods in TaskController
- Add tag management methods to apiService
- Create TagManager component for creating/deleting tags
- Update TaskForm to show tag selector with existing tags
- Display tags as colored badges in TaskItem component
- Add filter functionality in FilterBar to filter by tags

Success_criteria:
- All 6 tests pass
- Users can create custom tags with names and colors
- Tags can be added to and removed from tasks
- Multiple tags can be assigned to one task
- Tag filtering shows only tasks with selected tag
- Deleting a tag removes it from all tasks
- Tags persist across sessions
